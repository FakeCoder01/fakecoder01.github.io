Application Flow

1. Initialization

Load configuration, initialize map, detect user location or default to Moscow

2. Data Fetching

Fetch real-time flight data from OpenSky Network API based on current map bounds

3. Visualization

Display aircraft as pink markers on interactive map with smooth animations

4. Interaction

Handle user clicks, show flight details, manage multiple modal windows

5. Auto-Refresh

Automatically update flight data every 30 seconds for real-time tracking

6. Configuration

Apply user preferences and handle various display options and settings

Core Features Implementation

  • Real-time flight tracking with 30-second auto-refresh
  • Interactive Leaflet map with zoom, pan, and layer switching
  • GPS-based geolocation with fallback to Moscow coordinates

  • Modal system supporting up to 5 simultaneous windows
  • Responsive design with mobile-first approach
  • Error handling and offline detection
  • Keyboard shortcuts for power users
